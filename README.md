Arduino Day 2026 — TinkerSpace Kochi
Date: March 28, 2026
Venue: TinkerSpace, Kochi
Co-organised by: TinkerSpace & MakerGram

A full-day maker event where students, professionals, and hobbyists came together to learn, build, and share what's possible with Arduino. The day included talks, hands-on sessions, a robot competition, and a hardware unboxing — all in a beginner-friendly, community-driven atmosphere.
